# Redux Saga

## Configuração

- Para iniciar o servidor em desenvolvimento.

1. Instale todas as dependências:

   ```sh
   $ yarn install
   ```

2. Rode o projeto:

   ```sh
   $ yarn start
   ```

3. Abra [http://localhost:3000](http://localhost:3000) em seu navegador.

## Ferramentas utilizadas

- React
- Redux
- Redux Saga
