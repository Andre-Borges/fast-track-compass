# Petshop Alura

## Configuração

- Para iniciar o servidor em desenvolvimento.

1. Instale todas as dependências:

   ```sh
   $ yarn install
   ```

2. Inicie o servidor:

   ```sh
   $ yarn server
   ```

3. Rode o projeto:

   ```sh
   $ yarn start
   ```

4. Abra [http://localhost:3000](http://localhost:3000) em seu navegador.

## Ferramentas utilizadas

- React
- React Router Dom
- Json Server
